# Helsinki University MOOC — Java Programming I 🚀

This repository contains my solutions to the comprehensive **Java Programming I** course offered by the University of Helsinki (MOOC.fi). The course covers the fundamentals of software development, object-oriented programming (OOP), and algorithmic thinking using Java.

---

## 🛠️ Tech Stack & Concepts Covered

* **Language:** Java 
* **Concepts:** Object-Oriented Programming (OOP), Polymorphism, Encapsulation, Inheritance, Control Flow, Lists & Arrays, File I/O, Exception Handling, Unit Testing.
* **Tools Used:** VS Code, Test My Code (TMC) extension, Maven.

---

## 📂 Repository Structure

The project is structured into **7 distinct parts**, each focusing on intermediate to advanced core programming fundamentals:

* **Parts 1-3:** Basic logic, loops, methods, handling lists, and arrays.
* **Part 4:** Introduction to Object-Oriented Programming (Creating classes, managing objects).
* **Part 5:** Deep dive into OOP (Object references, overloaded constructors, keywords).
* **Part 6:** UI logic, architecture separating data from presentation, and testing.
* **Part 7:** Advanced algorithms, search/sort mechanics, and final comprehensive projects.

* **100% Core Mastery:** Every assignment in this repository has passed **100% of the local and server-side unit tests** verified by the MOOC.fi framework.
